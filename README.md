# Приложение для совместных поездок в офис 

Приложение должно помочь сотрудникам одной компании вместе ездить в офис.

Функциональность:

1. Отображение карты, на которой указаны офисы компании.
2. Водители могут выбирать офис в который поедут, место из которого поедут и время.
3. Отображение маршрута поездки и его примерно время.
4. Обычные пользователи могут посмотреть кто именно едет в необходимый офис и присоединиться к поездке.
5. Проработать ограничения, кто может присоединиться, а кто нет.

# Инструкция по запуску
Склонировать репозиторий:

```
git clone
```
Запустить приложение:

```
docker compose up -d
```

открыть браузер

- Thymeleaf, Leaflet, Spring Data JPA, Spring Security, Spring Web, Liquibase
